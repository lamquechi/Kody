How to format — no code to remember
  • Wrap a word in *asterisks* to give it a gentle emphasis.
  • Begin a line with >  to lift it into a pull-quote.
  • Begin a line with ##  to make it a chapter heading.
  • Put a single ✦ on its own line for a scene break.
  • Leave a blank line to start a new paragraph.

Or simply use the buttons above the page — they insert all of this for you.
